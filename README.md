# ObsidianOS

ObsidianOS is a simple, but powerful OS made in Scratch by [@keylimegreen541](https://scratch.mit.edu/users/keylimegreen541). It has not released yet, but dev releases are available. Please give feedback on ObsidianOS in the comments of the project on Scratch or the GitHub issue tracker. ObsidianOS uses the font Inter, and it requires it to be installed on your device. Press [here](https://fonts.google.com/specimen/Inter) to install it.
